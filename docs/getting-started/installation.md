# Installation

## Eungabi Core Module

you can simply use main functionalities of `Eungabi` navigation. 

=== "Groovy"

    ``` groovy
    implementation "io.github.easternkite:eungabi:<version>"
    ```

=== "Kotlin"

    ``` kotlin
    implementation("io.github.easternkite:eungabi:<version>")
    ```
